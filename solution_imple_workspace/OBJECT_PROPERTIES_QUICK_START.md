# 对象属性功能 - 快速开始

## 🎉 MVP版本已完成

对象属性功能的MVP版本已经完整实施，您现在可以在知识图谱中使用对象属性来描述关系的语义信息。

## ⚡ 5分钟快速体验

### 1️⃣ 启动应用

```bash
# 启动后端
cd backend
npm start

# 启动前端（新终端）
cd frontend
npm start
```

### 2️⃣ 导入测试数据

1. 打开浏览器访问 `http://localhost:3000`
2. 点击顶部"导入数据"按钮
3. 选择"JSON"标签
4. 上传文件：`data/object-properties-test-data.json`
5. 点击"导入"

### 3️⃣ 查看对象属性

**在图谱中：**
- 观察边的标签，会显示关键属性（如"优先级: 高"）

**在详情面板中：**
1. 点击任意节点（如 `vp-test-001`）
2. 查看右侧"对象属性关系"部分
3. 展开"出边关系"或"入边关系"

### 4️⃣ 编辑对象属性

1. 在对象属性关系中，点击"编辑"按钮
2. 修改属性值（如将优先级从"高"改为"中"）
3. 点击"确定"保存
4. 查看更新后的显示

## ❓ 常见问题

**Q: 为什么图谱中只显示部分对象属性？**  
A: 为避免图谱拥挤，边标签最多显示2个最重要的属性。完整属性请在详情面板中查看。

**Q: 如何添加新的对象属性？**  
A: 在 `core-domain-schema.json` 的 relationTypes 中添加 properties 定义即可。

**Q: 对象属性数据存储在哪里？**  
A: 存储在边数据的 `data` 字段中，格式：`{ "id": "e1", "source": "n1", "target": "n2", "type": "rel", "data": { "prop1": "value1" } }`
