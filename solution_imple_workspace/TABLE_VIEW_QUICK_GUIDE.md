# 📊 表格视图快速使用指南

## 🎉 新功能：属性数据表格展示

表格视图现在支持完整的节点属性和对象属性展示！

## ⚡ 3步快速体验

### 1️⃣ 切换到表格视图

点击顶部Header的 **"表格视图"** 按钮

### 2️⃣ 展开查看属性

**节点属性：**
- 切换到"节点"标签页
- 点击行前的 ▶️ 图标
- 查看完整的节点属性详情

**对象属性：**
- 切换到"关系/对象属性"标签页
- 点击行前的 ▶️ 图标
- 查看完整的对象属性详情

### 3️⃣ 搜索和过滤

- 在搜索框输入关键词
- 点击列标题的过滤图标
- 查看过滤后的结果

## 💡 使用技巧

### 快速展开所有行
暂不支持，使用"收起所有"按钮可以批量收起

### 搜索最佳实践
- 搜索节点ID：精确匹配
- 搜索类型：使用英文或中文标签
- 搜索属性值：直接输入值

### 优先级筛选
- 点击"优先级"列标题旁的过滤图标
- 勾选要查看的优先级（P0/P1/P2）
- 点击确定

### 类型筛选
- 点击"类型"或"关系类型"列的过滤图标
- 勾选要查看的类型
- 点击确定

## 🎨 视觉指南

### 颜色含义

**优先级标签：**
- 🔴 **红色** = P0（最高）
- 🟠 **橙色** = P1（高）
- 🔵 **蓝色** = P2（普通）

**状态标签：**
- 🟢 **绿色** = 正常状态
- 🟡 **黄色** = 警告
- 🔴 **红色** = 错误/阻塞

**类型标签：**
- 🔵 **蓝色** = 节点类型
- 🟢 **绿色** = 关系类型
- 🟣 **紫色** = 属性类型
- 🔷 **青色** = 统计信息

### 图标说明

- ▶️ **右箭头** - 点击展开
- ▼ **下箭头** - 点击收起
- 🔗 **蓝色链接** - 可点击跳转
- 🔍 **搜索图标** - 输入搜索
- 📊 **过滤图标** - 筛选数据

## 🚀 实战示例

### 示例1：查找所有P0需求

1. 切换到"节点"标签页
2. 点击"优先级"列的过滤图标
3. 勾选"P0"
4. 点击确定
5. 结果显示所有P0优先级的节点

### 示例2：查看特定关系的对象属性

1. 切换到"关系/对象属性"标签页
2. 在搜索框输入关系类型或ID
3. 找到目标关系行
4. 点击展开图标
5. 查看完整的对象属性

### 示例3：检查数据完整性

1. 切换到"节点"标签页
2. 查看"属性数量"列
3. 找到属性数量较少的节点
4. 展开查看缺失哪些属性
5. 返回图谱视图补充数据

### 示例4：追踪需求链路

1. 在表格中搜索Epic ID
2. 点击ID跳转到图谱视图
3. 查看完整的需求链路
4. 记录关键的关系ID
5. 返回表格查看关系的对象属性

## ❓ 常见问题

**Q: 如何一次展开多行？**  
A: 目前需要逐行点击展开。使用"收起所有"可以批量收起。

**Q: 搜索是否区分大小写？**  
A: 不区分，搜索会自动转换为小写匹配。

**Q: 可以编辑属性吗？**  
A: 当前版本仅支持查看。编辑请切换到图谱视图，在节点详情面板中编辑对象属性。

**Q: 如何导出表格数据？**  
A: 当前版本暂不支持导出。可以使用截图或复制数据。

**Q: 表格支持的最大数据量？**  
A: 理论上无限制，但建议单页不超过1000条以保证性能。使用分页和过滤来处理大数据量。

**Q: 展开行的数据会实时更新吗？**  
A: 是的，数据共享相同的数据源，刷新数据后展开行也会更新。
