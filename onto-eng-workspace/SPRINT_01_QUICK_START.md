# Sprint 01 快速开始指南

## 🚀 快速启动

### 1. 启动后端服务
```bash
cd backend
npm start
```
后端服务将运行在 `http://localhost:3001`

### 2. 启动前端应用
```bash
cd frontend
npm start
```
前端应用将运行在 `http://localhost:3000`

## 🎯 新功能概览

### 5种视图模式

在顶部导航栏的Segmented控件中切换视图：

1. **🔹 图谱视图** - 网络关系可视化（已有功能）
2. **📋 表格视图** - 结构化数据表格（已有功能）
3. **🌲 树形视图** - 类层次和实例树（✨新增）
4. **🔥 矩阵视图** - 关系矩阵热力图（✨新增）
5. **📊 仪表盘视图** - 统计分析仪表盘（✨新增）

---

## 📖 使用教程

### 树形视图 🌲

**适用场景**: 浏览本体的层次结构，快速定位实例

**操作步骤**:
1. 点击顶部"树形"按钮
2. 切换Tab查看"类层次"或"实例树"
3. 使用搜索框过滤节点
4. 点击"全部展开/折叠"控制显示层级

**提示**: 
- 类层次树显示实体类型的继承关系
- 实例树按类型分组显示所有节点
- 括号中的数字表示实例数量

---

### 矩阵视图 🔥

**适用场景**: 分析节点间的关系模式和强度

**操作步骤**:
1. 点击顶部"矩阵"按钮
2. 选择要查看的关系类型（或"所有关系"）
3. 使用"显示数值"开关控制是否显示关系数量
4. 鼠标悬停在热力图上查看详细信息

**提示**:
- 颜色越深表示关系越多
- 可以通过关系类型过滤专注分析特定关系
- 适合识别关系密集的节点对

---

### 仪表盘视图 📊

**适用场景**: 获取知识图谱的整体统计和分析

**功能模块**:
1. **核心指标**：节点总数、关系总数、平均度数、图密度
2. **节点类型分布图**：柱状图显示各类型实例数量
3. **关系类型分布图**：环形饼图显示关系类型占比
4. **核心节点排名**：Top 5 连接数最多的节点
5. **Schema统计**：实体类型和关系类型定义

**提示**:
- 平均度数反映图的连通性
- 图密度范围[0, 1]，值越大表示图越密集
- 核心节点通常是图中的关键实体

## 💡 使用技巧

### 技巧1：组合视图使用
1. 先在**仪表盘**了解整体情况
2. 在**树形视图**找到感兴趣的实例
3. 切换到**图谱视图**查看该节点的网络关系
4. 使用**矩阵视图**分析关系模式

### 技巧2：快速数据分析
1. **仪表盘**查看核心指标和Top节点
2. **矩阵视图**识别关系密集区域
3. **图谱视图**深入探索具体关系

### 技巧3：大数据处理
- 矩阵视图：使用关系类型过滤减少数据量
- 树形视图：使用搜索功能快速定位
- 表格视图：使用筛选和排序功能

## 🔧 故障排查

### 问题1：视图切换后显示空白
**解决方案**: 检查控制台是否有错误，刷新页面重试

### 问题2：矩阵视图加载很慢
**解决方案**: 
- 使用关系类型过滤减少数据量
- 等待渲染完成（大规模数据可能需要2-3秒）

### 问题3：图表显示不完整
**解决方案**: 
- 调整浏览器窗口大小
- 切换到其他视图再切换回来
