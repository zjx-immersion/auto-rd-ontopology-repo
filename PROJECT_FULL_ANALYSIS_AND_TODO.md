## 项目概述

### 基本信息

| 属性 | 内容 |
|------|------|
| **项目名称** | 本体图谱工程平台 (Ontology Graph Engineering Platform) |
| **当前应用** | 岚图汽车研发体系知识图谱 |
| **技术栈** | React 18 + Node.js + Express + JSON文件存储 |
| **当前版本** | v2.0 |
| **最后更新** | 2026-02-11 |

### 项目定位

基于本体建模的可视化知识图谱工程平台，针对不同领域/场景能够快速进行：
- 本体建模 (Schema Editor)
- 可视化展现
- 基于 Schema 实例化生成 OAG 图谱
- 图谱 Agent 助手提供智能能力
- 为外部 Agent 提供 API 服务
